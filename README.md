# Regulating ChatGPT: Interactive Multimodal Project

This project is now implemented as a Vite React app. It presents the thesis that large language models such as ChatGPT should be regulated through a flexible, risk-based approach rather than a universal one-size-fits-all model.

## Files

- `index.html`: Vite HTML shell
- `src/main.jsx`: mounts the React app
- `src/App.jsx`: React components and interactive logic
- `src/styles.css`: visual design, layout, and responsive behavior
- `package.json`: project scripts and dependencies
- `vite.config.js`: Vite configuration

## Interactive features included

- Audience poll in the hero section
- Clickable scenario explorer with dynamic risk and oversight meters
- Policy simulator with context selector, oversight slider, and safeguard toggles
- Risk calculator with a visual score dial
- Quiz for live presentation use

## Notes

- The app uses local React and Vite dependencies.
- Run `npm install` once, then use `npm run dev`.

## Presentation angle

This site is designed to help you explain how the multimodal form supports the argument:

- The scenario explorer shows that risk depends on context.
- The policy lab visualizes tradeoffs between innovation, safety, and accountability.
- The calculator turns research into a decision framework for the audience.
- The quiz and poll increase audience participation during the presentation.

## How to use

Run `npm install` and then `npm run dev`.
